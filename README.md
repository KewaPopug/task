## Описание

Проект в котором реализована клиентская часть и страница администратора. Проект предоставляет API, работающее поверх HTTP в формате JSON.

## Требования

В процессе создания проекта использованы следующие технологии:

- PHP (7.4)
- Yii (2.0.48)
- PostgresSQL (14)
- Composer (2.3.10)
- Apache (2.4.52)
- Ubuntu (22.04)
